# 化妆品销售商城系统 python1284

## 项目概述


## 技术栈

- 后端：Python 3.7/3.8，Django 框架
- 数据库：MySQL 5.7 或更高版本
- 前端：Vue.js

## 功能模块

系统主要包含以下功能模块：

1. 用户模块
   - 用户注册与登录
   - 用户信息管理
   - 用户订单管理

2. 商品模块
   - 商品展示
   - 商品分类
   - 商品搜索

3. 购物车模块
   - 添加商品至购物车
   - 修改购物车商品数量
   - 删除购物车商品

4. 订单模块
   - 提交订单
   - 订单支付
   - 订单状态跟踪

5. 数据分析模块
   - 销售数据分析
   - 用户行为分析

## 系统角色

- 普通用户：浏览商品、购买商品、管理个人账户
- 管理员：管理商品、订单、用户及数据分析

## 运行环境

- 后端：Python 3.7/3.8，Django 框架
- 数据库：MySQL 5.7 或更高版本
- 前端：Vue.js，现代浏览器

## 部署步骤

1. 安装 Python 3.7/3.8
2. 安装 MySQL 数据库，版本 5.7 或更高
3. 克隆项目代码，安装项目依赖
4. 配置数据库连接信息
5. 运行项目

## 目录结构

```
项目名称/
|-- backend/                # 后端代码目录
|   |-- apps/               # 应用代码目录
|   |-- config/             # 配置文件目录
|   |-- manage.py           # 项目管理脚本
|-- frontend/               # 前端代码目录
|   |-- src/                # Vue.js 源代码目录
|-- static/                 # 静态文件目录
|-- requirements.txt        # 项目依赖文件
```
